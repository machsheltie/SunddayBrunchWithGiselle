# Development Guide: sunday-brunch-website

This guide provides instructions for setting up and running the `sunday-brunch-website` part of the project.

## Prerequisites

- Node.js
- npm

## Installation

```bash
npm install
```

## Environment Setup

1.  Copy `.env.example` to `.env`.
2.  Sign up for a free [ConvertKit](https://convertkit.com) account.
3.  Get your Form ID and API Key.
4.  Update the `.env` file with your ConvertKit credentials.

## Running the Development Server

```bash
npm run dev
```

## Building for Production

```bash
npm run build
```
