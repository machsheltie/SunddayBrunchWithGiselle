# Project Overview

## Project Name

Sunday Brunch With Giselle

## Executive Summary

This project consists of two separate web applications: `sunday-brunch-website` and `website`. Both are built with React and Vite. `sunday-brunch-website` appears to be the main application and includes components for an episode player, recipe cards, and character callouts. The `website` part is a more minimal setup.

## Technology Stack Summary

- **Primary Framework**: React
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: CSS, Tailwind CSS (in `website`)

## Architecture

- **Repository Structure**: Multi-part
- **Architecture Pattern**: Component-based
