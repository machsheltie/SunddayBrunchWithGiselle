# EPIC: Launch "Sunday Brunch With Giselle" Podcast & Brand

**Goal:** Create and launch a cozy, whimsical podcast and recipe blog featuring Stacey and her Sheltie, Giselle (voiced by AI), to provide a sanctuary of warmth, storytelling, and baking for listeners.

---

## PHASE 1: Foundation & Brand Identity (Weeks 1-2)
**Objective:** Define the visual and tonal identity of the brand and prepare the initial assets.

- [ ] **Brand Definition**
    - [ ] Finalize Podcast Title: "Sunday Brunch With Giselle" (or similar).
    - [ ] Define Tagline: e.g., "Whimsy, Warmth & Wags" or "A Cozy Podcast for Grownups Who Still Believe in Magic."
    - [ ] Create Brand Style Guide:
        - [ ] Color Palette: Soft, warm, cozy tones (e.g., sage green, buttercream yellow, terracotta).
        - [ ] Typography: Readable serif for body, whimsical script for headers.
        - [ ] Voice/Tone: Warm, empathic, witty, slightly mischievous (Giselle).
- [ ] **Visual Assets**
    - [ ] Design Podcast Cover Art: Featuring Stacey and Giselle (illustration or stylized photo).
    - [ ] Create Social Media Profile Images & Banners (Instagram, YouTube, Facebook, TikTok).
    - [ ] Design "Giselle" Character Visuals: For use in social posts and potentially video overlays.
- [ ] **Platform Setup**
    - [ ] Secure Social Media Handles: Instagram, TikTok, YouTube, Facebook.
    - [ ] Register Domain Name (if applicable).

## PHASE 2: Technical Setup & Production Pipeline (Weeks 1-2)
**Objective:** Set up the necessary tools and workflows for high-quality audio and content production.

- [ ] **Audio Tools Setup**
    - [ ] **Giselle's Voice (ElevenLabs):**
        - [ ] Create ElevenLabs account.
        - [ ] Select or train a voice model for Giselle (Posh, sassy, refined).
        - [ ] Test voice with sample scripts to refine emotion and timing.
    - [ ] **Recording & Editing:**
        - [ ] Set up DAW (GarageBand, Audacity, or Descript).
        - [ ] Configure Microphone (Blue Yeti/Shure MV7) and recording environment.
        - [ ] Create an "Intro/Outro" template with theme music.
- [ ] **Website Foundation**
    - [ ] Select Platform: WordPress, Squarespace, or Custom (React/Vite).
    - [ ] Create Landing Page: "Coming Soon" with email signup.
    - [ ] Set up Blog Structure: Categories for Recipes, Episodes, Art, Garden.

## PHASE 3: Content Production (Weeks 3-4)
**Objective:** Produce the pilot episode "The Pie That Started a Dynasty" and create the accompanying website content.

- [ ] **Episode 1: French Silk Pie**
    - [ ] **Audio Production:**
        - [ ] Generate Voiceovers: Giselle (Regal/Sassy), Havok (Military), Phaedra (Science), Tiana (Joy).
        - [ ] Record Host Audio: Stacey's segments (Intro, Story, Recipe, Outro).
        - [ ] Assemble Audio: Mix voiceovers, host audio, and sound effects (kitchen sounds, music).
    - [ ] **Website Content:**
        - [ ] Create **Episode Page Template**: Audio player, show notes, transcript.
        - [ ] Create **Recipe Component**: "French Silk Pie" with printable view and "Sheltie Tips".
        - [ ] Implement **"Meet the Cast"** section on the About page using the Persona details.

## PHASE 4: Soft Launch & Community Building (Weeks 5-6)
**Objective:** Release the pilot and build the "Cozy Sanctuary" community.

- [ ] **Launch Strategy**
    - [ ] Upload Episode 1 to hosting platform.
    - [ ] Publish "The Pie That Started a Dynasty" blog post.
    - [ ] Tease "Next Week: Cinnamon Rolls" on social media.
- [ ] **Community**
    - [ ] Launch "Sunday Letter" newsletter.
    - [ ] Engage with listeners on the "Whimsy, Warmth & Wags" theme.

## PHASE 5: Growth & Expansion (Ongoing)
**Objective:** Grow the audience and explore monetization.

- [ ] **Marketing**
    - [ ] Cross-promote with other cozy/art/pet creators.
    - [ ] Optimize SEO for the blog (Recipes, Podcast transcripts).
- [ ] **Monetization**
    - [ ] Launch Patreon/Substack for bonus content.
    - [ ] Explore Merch (Aprons, Mugs, Prints).
    - [ ] Seek Sponsorships (Tea, Baking supplies, Art supplies).
