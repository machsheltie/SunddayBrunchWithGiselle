# Typography Reference Guide

## Font Acquisition & Usage

### Display/Headers: Migra
**Why Migra?**
- Contemporary serif with very soft curves
- Avoids the traditional, overused serif look
- Perfect balance of modern and warm

**Where to Find:**
- Search for "Migra font" on font marketplaces (Creative Market, MyFonts, FontSpring)
- License required for commercial use

**Usage:**
- Podcast episode titles
- Main website headers
- Social media graphics titles

---

### Script/Accent: Manekin
**Why Manekin?**
- Bold vintage script with character
- 40+ stylistic alternates for variety
- Multi-language support

**Where to Find:**
- Available on Creative Market, Envato Elements
- License required for commercial use

**Usage:**
- Tagline ("Whimsy, Warmth & Wags")
- Pull quotes
- Special accents and decorative elements

---

### Body Text: Spezia Serif
**Why Spezia Serif?**
- Classic, calm feeling
- Strong readability for long-form content
- Premium feel without being loud

**Where to Find:**
- Search font marketplaces for "Spezia Serif"
- License required for commercial use

**Usage:**
- Blog post body text
- Episode descriptions
- Recipe instructions

---

## Fallback Options
If primary fonts are unavailable or budget-constrained:
- **Headers:** Rustic or Faxi
- **Script:** Onthel or Popless Script
- **Body:** Larken

## Free Alternatives (If Needed)
While not as distinctive, these Google Fonts can serve as temporary placeholders:
- **Headers:** Cormorant Garamond (elegant serif)
- **Script:** Pacifico (casual script)
- **Body:** Crimson Text (readable serif)
